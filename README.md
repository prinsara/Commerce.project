# Commerce Project

## 프로젝트 소개
Java로 구현한 간단한 커머스 플랫폼 프로젝트입니다.  
카테고리별로 상품을 관리하고, 사용자가 카테고리와 상품을 선택해 정보를 확인할 수 있도록 구현했습니다.

## 주요 기능
- 카테고리 목록 출력
- 카테고리 번호 입력 처리
- 선택한 카테고리의 상품 목록 출력
- 상품 번호 입력 처리
- 선택한 상품 상세 정보 출력
- `0` 입력 시 종료 및 뒤로가기 처리
- 잘못된 번호 입력 시 안내 메시지 출력

## 실행 방법
1. `Main.java`를 실행합니다.
2. 메인 화면에서 원하는 카테고리 번호를 입력합니다.
3. 선택한 카테고리의 상품 목록을 확인합니다.
4. 원하는 상품 번호를 입력합니다.
5. 선택한 상품의 상세 정보를 확인합니다.
6. `0`을 입력하면 이전 화면으로 돌아가거나 프로그램을 종료할 수 있습니다.

## 파일 구조
- `Main.java` : 상품과 카테고리 객체를 생성하고 프로그램을 시작하는 클래스
- `Product.java` : 상품명, 가격, 설명, 재고수량을 관리하는 클래스
- `Category.java` : 카테고리명과 상품 목록을 관리하는 클래스
- `CommerceSystem.java` : 입력, 출력, 반복문, 선택 흐름을 처리하는 클래스

## 클래스 역할
- `Product`
  - 개별 상품 정보를 저장합니다.
- `Category`
  - 같은 종류의 상품들을 하나의 카테고리로 묶어 관리합니다.
- `CommerceSystem`
  - 사용자 입력을 받고 프로그램 흐름을 제어합니다.
- `Main`
  - 프로그램 실행을 위한 객체를 생성하고 시작합니다.

## 예외 처리
- 카테고리 선택 시 범위를 벗어난 숫자 입력 방지
- 상품 선택 시 범위를 벗어난 숫자 입력 방지
- `0` 입력 시 종료 또는 뒤로가기 처리

## 프로젝트를 통해 배운 점
- `Product`, `Category`, `CommerceSystem`으로 역할을 나누며 객체 지향 구조를 연습할 수 있었습니다.
- 리스트에서 사용자가 선택한 번호에 맞는 데이터를 꺼내어 다음 단계로 연결하는 흐름을 구현해볼 수 있었습니다.
